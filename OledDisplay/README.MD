# What

Generic Mini I2C 128x64 OLED Display with a SSD1306 controller. There are tons out there. I used this one: 

https://www.amazon.co.uk/gp/product/B076PNP2VD

# How to control it? 

https://github.com/olikraus/u8g2

# Pins

* A4 -> SDA
* A5 -> SCL
* 5V -> VCC
* GND -> GND
